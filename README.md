## FoundIt!

- FoundIt! is a lost&found platform created for my college because it didn't had any unified structure to list lost or found items in college campus.
- I've used AWS S3, DynamoDB, Next.js Serverless for Backend.
- ShadCN, TailwindCSS and Next.js for Frontend.
